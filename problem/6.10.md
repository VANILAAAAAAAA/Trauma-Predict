# 6.10 — Field Construction Issues to Confirm

第一轮只列 5 个。目标是尽快确认能否进入当前 field / static-hour-day 设计。

| # | Issue | Current handling | Need confirmation |
|---:|---|---|---|
| 1 | `head_injury` 是否加入 STATIC | 用 MIMIC-IV `diagnoses_icd` 构建；不使用 E-code 作为头部解剖来源。推荐主规则：ICD-10 `S00-S09` 或 ICD-9 `800-804, 850-854`；strict TBI sensitivity：ICD-10 `S06` 或 ICD-9 `850-854`。 | 主线用 broad head injury 还是 strict TBI？如果作为模型 STATIC input，是否接受 discharge ICD 的 retrospective 性质；否则作为 stratification/ablation。 |
| 2 | `rSI` / shock index 定义 | MIMIC 可由 ED HR 和 ED SBP 构建。当前不确定 UW 的 `rSI` 是 `HR/SBP` 还是 reverse SI `SBP/HR`。 | 是否把字段命名拆开：`si = HR/SBP`，`rsi = SBP/HR`，两者都构建；主线只等 UW 定义确认后选择。 |
| 3 | `initial_ed_sbp` 与 ED linkage 缺失 | 有 ED triage/vitalsign 时取 ED SBP；无 ED linkage 时保持 `NA`，不使用 ICU 首个 SBP 替代。 | 是否接受 `ed_sbp-NA` 作为 STATIC 缺失；还是允许用 admission/ICU first SBP 做 fallback，并显式标记 source。 |
| 4 | `strong_ion` 计算与 recency | 公式：`StrongIon = (Na + K) - (Cl + bicarb)`。小时级用 latest-at-or-before-h 的 Na/K/Cl/bicarb 计算，同时保留最大 recency。 | 如果四个成分 recency 差距很大，是否仍输出 `strong_ion`；是否设置最大 recency 阈值，如 >24h 则 `NA`。 |
| 5 | `lymphocytes` / `neutrophils` 单位 | UW 分布更像 absolute count，单位按 `K/uL`。MIMIC 若只有 differential `%`，当前 sample 用同时间 WBC × % / 100 转成 K/uL。 | 是否接受该转换作为主线规则；还是只使用 MIMIC absolute count itemids，百分比样本不转、直接置缺失。 |

## Current recommendation

1. `head_injury`: broad rule 进 STATIC，但同时保存 `head_injury_rule=broad/strict` 和 source ICD provenance。
2. `rSI`: 先同时构建 `si` 和 `rsi_reverse`，等 UW 定义确认后再进 include-only。
3. `initial_ed_sbp`: 缺 ED linkage 就保持 `NA`，不要混用 ICU SBP。
4. `strong_ion`: 输出值，但带 recency；是否阈值截断待确认。
5. `lymph/neut`: 主线接受 WBC × differential % 转换，但标记 `derived_from_percent`。
